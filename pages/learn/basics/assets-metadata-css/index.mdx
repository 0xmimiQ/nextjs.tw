我們剛剛新增的第二個頁面目前沒有樣式。讓我們添加 CSS 來為該頁面美化樣式。

Next.js 有內建對 <a href="/docs/basic-features/built-in-css-support" className="link-code" target="_blank" >CSS</a> 和 <a href="/docs/basic-features/built-in-css-support#sass-support" className="link-code" target="\_blank" >Sass</a> 的支援。 在這個課程，我們將會使用 CSS。

這堂課程還將介紹 Next.js 是如何處理靜態資產，像是圖片、頁面中的詮釋資料（Metadata），例如 `<title>` 標籤。

### 你將會在這課程學到什麼

在這課程，你將學到 :

- 如何在 Next.js 中添加<a href="/docs/basic-features/static-file-serving"  className="link-code" target="_blank">靜態文件</a>(例如圖像)。
- 如何自定義每個頁面 `<head>` 中的內容。
- 如何創建一個可重用的 React 組件，並使用 <a href="/docs/basic-features/built-in-css-support#adding-component-level-css" className="link-code" target="_blank"> CSS 模塊</a>進行樣式設計。
- 如何在 `pages/_app.js` 中添加全局 CSS。
- 一些在 Next.js 中設計樣式的小技巧。

### 須先具備

    - 基礎的 CSS 知識。本課程將介紹如何在 Next.js 應用程式中加入 CSS，但不會涵蓋 CSS 基礎知識。

> 如果你正在尋找 Next.js 樣式的詳細文件，歡迎查看<a href="/docs/basic-features/built-in-css-support" target="_blank">CSS 文件</a>
