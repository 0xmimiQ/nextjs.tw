#### 設置
首先，讓我們確保您的開發環境已準備就緒。

- 如果您沒有安裝**Node.js**，請從[此處][1]安裝。您需要 Node.js 版本**10.13**或更高版本。
- 您將在本教學中使用自己的文字編輯器和終端應用程式。

<Blockquote>
如果您作業系統使用的是 Windows，我們建議您[下載 Git for Windows][2]並使用它附帶的 Git Bash，它支援本教學中特定於 UNIX 的命令。[適用於 Linux 的 Windows 子系統 (WSL)][3]是另一種選擇。
</Blockquote>

#### 創建 Next.js 應用
要創建 Next.js 應用程式，請打開終端，`cd`進入要在其中創建應用式的目錄，然後執行以下指令：

```shell
npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/master/basics/learn-starter"
```
<Blockquote>
在後台，它使用[`create-next-app`][4]工具，引導您創建 Next.js 應用程式。它通過`--example`參數使用[這個模板][5]。

如果它不起作用，請查看[此頁面][6]。
</Blockquote>

運行開發服務器
您現在有一個名為nextjs-blog. 讓我們cd進入它：
```properties
cd nextjs-blog
```

然後，運行以下命令：

npm run dev
這會在端口3000上啟動 Next.js 應用程序的“開發服務器”（稍後會詳細介紹） 。

讓我們檢查一下它是否有效。從瀏覽器打開http://localhost:3000 。
