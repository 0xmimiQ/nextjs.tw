我們要來建立一個部落格（這是<a href="https://next-learn-starter.vercel.app/" target="_blank">預想的結果</a>），但是目前為止我們還沒加入任何有關部落格的內容。在這次的課程中，我們將學習如何在應用程式中獲取外部的部落格資料。我們會將部落格的內容存在檔案系統，但如果資料的儲存在其它地方（例如：<a href="https://en.wikipedia.org/wiki/Headless_content_management_system" target="_blank">Headless CMS</a> 的資料庫）我們的部落格也會正常運作。


#### 你將在這課程中學到什麼

在這課程中，你將學到：
- Next.js 的<a href="/docs/basic-features/pages" target="_blank">提前渲染</a>功能
- 兩種不同形式的提前渲染：<a href="/docs/basic-features/pages" target="_blank">靜態生成</a>和<a href="/docs/basic-features/pages" target="_blank">伺服器端渲染</a>
- <a href="/docs/basic-features/pages" target="_blank">使用資料</a>的靜態生成和<a href="/docs/basic-features/pages" target="_blank">不使用資料</a>的靜態生成
- 如何使用 <a href="/docs/basic-features/data-fetching/overview" target="_blank">`getStaticProps`</a> 將外部的部落格資料載入首頁
- 一些關於 <a href="/docs/basic-features/data-fetching/overview" target="_blank">`getStaticProps`</a> 的實用資訊
