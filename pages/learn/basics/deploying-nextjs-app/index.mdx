在這個最後的基礎課程，我們將會部署 Next.js 程式到正式環境。

我們會學習如何部署 Next.js 到 Vercel - 一個由 Next.js 開發者們所建構的平台。我們也會提到其他的開發選項。

#### 你將在這課程中學到什麼

在這課程中，你將學到：
- 如何將你的 Next.js 程式部署到 <a href="https://vercel.com/?utm_source=next-site&utm_medium=learnpages&utm_campaign=next-website" target="_blank">Vercel</a> 。
- 學習 **DPS** 開發流程： **D**evelop（開發），**P**review（預覽），**S**hip（部署上線）。
- 如何部署你的 Next.js 程式到你選擇的網頁託管提供者
