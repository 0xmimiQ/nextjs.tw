在這一堂課，你會學到:

- 如何藉由[動態路由](/docs/routing/dynamic-routes)使用 [`getStaticPaths`](/docs/basic-features/data-fetching#getstaticpaths-static-generation)來靜態產生頁面。
- 如何在每一個部落格文章中撰寫 [`getStaticPaths`](/docs/basic-features/data-fetching#getstaticprops-static-generation) 來抓取文章資料。
- 如何使用 [`remark`](https://github.com/remarkjs/remark) 渲染 markdown 語法。
- 如何美化日期字串。
- 如何使用[動態路由](/docs/routing/dynamic-routes)導航網頁。
- 一些有用的[動態路由](/docs/routing/dynamic-routes)資訊。
