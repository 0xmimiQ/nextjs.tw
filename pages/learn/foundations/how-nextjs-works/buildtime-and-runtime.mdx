#### 構建時間和運行時間
構建時間（或構建步驟）是為生產準備應用程序代碼的一系列步驟的名稱。

當您構建應用程序時，Next.js 會將您的代碼轉換為生產優化文件，準備好部署到服務器並供用戶使用。這些文件包括：

+  靜態生成頁面的 HTML 文件
+  用於在服務器上呈現頁面的JavaScript 代碼
+  用於使頁面在客戶端上交互的 JavaScript 代碼
+  CSS 文件
運行時間（或請求時間）是指在構建和部署應用程序之後，應用程序響應用戶請求而運行的時間段。
