#### 構建時間和運行時間
**構建時間**（或構建步驟）是為生產準備應用程序程式碼的一系列步驟的名稱。

當您構建應用程序時，Next.js 會將您的程式碼轉換為生產優化文件，準備好部署到服務器並供用戶使用。這些文件包括：

+  靜態生成頁面的 HTML 檔案
+  用於在伺服器上渲染頁面的 JavaScript 程式碼
+  用於在客戶端上使頁面具有互動性的 JavaScript 程式碼
+  CSS 檔案

**運行時間**（或請求時間）是指在構建和部署應用程式之後，應用程式響應用戶請求而運行的時間段。
