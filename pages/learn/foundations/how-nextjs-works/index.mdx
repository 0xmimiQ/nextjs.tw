在您學習更進階的 Next.js 功能前，了解 Next.js 運作的基礎原理會很有幫助。

在本課程的開始，我們討論了對於您如何建置和構建您的應用程式 React 是如何相對地自由 - 有多種方式可使用 React 來建置應用程式。Next.js 提供了一個框架來構建您的應用，可有助於加速開發過程和最終應用程式的優化。

在接下來的部分中，我們將介紹在這些不同階段裡，您的應用程式碼會發生什麼事：

- 您的程式碼在 **何環境** 運行：**開發環境 vs. 正式環境**
- 您的程式碼在 **何時** 運行：**建置時 vs. 運行時**
- 渲染在 **何處** 發生：**客戶端 vs. 伺服器**

現在，讓我們更深入地研究這些概念，並討論 Next.js 在幕後所執行的一些過程。
