在學習更進階的 Next.js 內容前，先了解 Next.js 的運作原理會對你很有幫助。

在本課程的開始，我們談到 React 並不太在意你如何去構建你的應用程式，你有多種不同的方法來使用 React 進行開發。 Next.js 提供了一個框架可主導你構建應用，並且能降低開發過程的時間成本及優化應用程式的性能。

在接下來的部分中，我們將介紹在這些不同階段裡，你的應用程式碼會發生什麼事：

- 你的程式碼在 **何環境** 運行：**開發環境 vs. 正式環境**
- 你的程式碼在 **何時** 運行：**建置時 vs. 運行時**
- 渲染在 **何處** 發生：**客戶端 vs. 伺服器**

現在，讓我們更深入地研究這些概念，並討論 Next.js 在幕後所執行的一些過程。
